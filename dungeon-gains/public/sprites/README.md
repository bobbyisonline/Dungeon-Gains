# Sprite Sheets

## Sword Sprites

**common-swords.png** - 4x4 grid, 64x64 pixels per sprite

Grid layout (0-indexed):
- Row 0: Swords 0-3
- Row 1: Swords 4-7
- Row 2: Swords 8-11
- Row 3: Swords 12-14 (slot 15 is empty)

## Usage

Sprites use CSS background-position to display specific weapons from the sheet.
